# contoso-traders-devsecops-github-hackathon
Accelerate DevSecOps mastery with the Contoso Traders e-commerce app (React, .NET, microservices). This hackathon repo guides you through CI/CD with GitHub Actions, GitHub Advanced Security (CodeQL, Dependabot, secret scanning), Azure Monitor &amp; App Insights integration, resilience testing, and AI-powered development with GitHub Copilot.
